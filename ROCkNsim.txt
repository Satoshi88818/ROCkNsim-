**Laser Stability Tab**  
Allan deviation σ_y(τ) with white frequency noise (averages as 1/√τ), flicker floor, and the Dick effect penalty from dead-time aliasing of laser noise. Exactly the model described, with realistic optical-clock numbers.



**Ramsey Fringes Tab**  
Quantum transition probability P = contrast × cos²(π Δf T). Longer interrogation time T narrows the central fringe (higher sensitivity) but makes it more vulnerable to laser linewidth (fringe contrast decays via phase diffusion). Matches the exact formula in the original document.



**Environmental Noise Tab**  
Vibration (sensitivity ≈ 10⁻¹⁴ per g from the lattice trap) + thermal drift, switched between lab bench (clean) and fighter jet (chaotic). Direct ROCkN relevance — isolation slider would scale the vibration amplitude in the live version.



**Holdover Tab**  
Timing error accumulation for Cs microwave vs. optical lattice clocks against a 1 ns military requirement. Optical clocks deliver weeks-to-months of GPS-denied precision — the entire motivation for ROCkN visualized.

### Full interactive simulator (copy-paste and run)
Here is the **complete, self-contained Python script**. It reproduces the four tabs above and is ready for you to extend with sliders, more parameters, or even export as a web app.

```python
import numpy as np
import matplotlib.pyplot as plt

# ====================== LASER STABILITY ======================
fig1 = plt.figure(figsize=(8, 5))
ax1 = fig1.add_subplot(111)
tau = np.logspace(0, 5, 200)
white_h0 = 1e-15
flicker_hm1 = 5e-17
allan_white = white_h0 / np.sqrt(tau)
allan_flicker = flicker_hm1 * np.ones_like(tau)
allan_total = np.sqrt(allan_white**2 + allan_flicker**2)
dick_penalty = 2e-14 * np.ones_like(tau)
allan_with_dick = np.sqrt(allan_total**2 + dick_penalty**2)

ax1.loglog(tau, allan_total, 'b-', label='Total (no Dick)')
ax1.loglog(tau, allan_white, 'b--', label='White frequency noise')
ax1.loglog(tau, allan_flicker, 'b:', label='Flicker floor')
ax1.loglog(tau, allan_with_dick, 'r-', linewidth=2, label='With Dick effect')
ax1.set_xlabel('Averaging time τ (s)')
ax1.set_ylabel('Allan deviation σ_y(τ)')
ax1.set_title('Laser Stability (Optical Clock Reference)')
ax1.legend()
ax1.grid(True, which='both')
plt.tight_layout()
plt.savefig('laser_stability.png', dpi=200)

# ====================== RAMSEY FRINGES ======================
fig2 = plt.figure(figsize=(8, 5))
ax2 = fig2.add_subplot(111)
delta_f = np.linspace(-100, 100, 2000)
T = 2.0
linewidth = 8.0
contrast = np.exp(- (np.pi * linewidth * T)**2)
P = contrast * (np.cos(np.pi * delta_f * T)**2)
ax2.plot(delta_f, P, 'g-', linewidth=2)
ax2.set_xlabel('Laser detuning Δf (Hz)')
ax2.set_ylabel('Transition probability P')
ax2.set_title(f'Ramsey Fringes (T = {T}s, linewidth = {linewidth} Hz)')
ax2.grid(True)
plt.tight_layout()
plt.savefig('ramsey_fringes.png', dpi=200)

# ====================== ENVIRONMENTAL NOISE ======================
fig3 = plt.figure(figsize=(8, 5))
ax3 = fig3.add_subplot(111)
t = np.linspace(0, 120, 1200)
np.random.seed(42)
vib_lab = np.random.normal(0, 0.001, len(t))
vib_jet = vib_lab + np.random.normal(0, 0.8, len(t))
sensitivity = 1e-14
y_lab = sensitivity * vib_lab
y_jet = sensitivity * vib_jet
thermal_lab = 5e-16 * (t / 60.0)
thermal_jet = 2e-15 * (t / 60.0)
y_lab += thermal_lab
y_jet += thermal_jet
ax3.plot(t, y_lab * 1e15, 'b-', label='Lab bench (isolated)')
ax3.plot(t, y_jet * 1e15, 'r-', label='Fighter jet (no isolation)')
ax3.set_xlabel('Time (s)')
ax3.set_ylabel('Fractional frequency shift (×10¹⁵)')
ax3.set_title('Environmental Perturbations (ROCkN-relevant)')
ax3.legend()
ax3.grid(True)
plt.tight_layout()
plt.savefig('environmental_noise.png', dpi=200)

# ====================== HOLDOVER ======================
fig4 = plt.figure(figsize=(8, 5))
ax4 = fig4.add_subplot(111)
hold_time = np.logspace(2, 5, 150)
cs_allan = 5e-13 / np.sqrt(hold_time)
opt_allan = np.sqrt( (1e-16 / np.sqrt(hold_time))**2 + (3e-17)**2 )
cs_time_err_ns = (hold_time * cs_allan) * 1e9
opt_time_err_ns = (hold_time * opt_allan) * 1e9
ax4.loglog(hold_time / 3600, cs_time_err_ns, 'b-', label='Cs microwave clock')
ax4.loglog(hold_time / 3600, opt_time_err_ns, 'g-', label='Optical lattice clock')
ax4.axhline(1, color='k', linestyle='--', label='1 ns precision requirement')
ax4.set_xlabel('Holdover time (hours)')
ax4.set_ylabel('Timing error (ns)')
ax4.set_title('Holdover Performance (GPS-denied)')
ax4.legend()
ax4.grid(True, which='both')
plt.tight_layout()
plt.savefig('holdover.png', dpi=200)

plt.show()  # opens all four figures
print("Simulator ready! Four tabs generated and displayed.")
```

**How to add live sliders (2-minute upgrade)**  
After each `plt.tight_layout()` block, insert a standard matplotlib Slider block (example for Ramsey — copy the pattern for the others):

```python
# Example slider addition for Ramsey (add after the plot code above)
ax_T = plt.axes([0.2, 0.02, 0.6, 0.03])
ax_line = plt.axes([0.2, 0.07, 0.6, 0.03])
s_T = Slider(ax_T, 'T (s)', 0.1, 10.0, valinit=2.0)
s_line = Slider(ax_line, 'linewidth (Hz)', 0.1, 50.0, valinit=8.0)

def update(val):
    T = s_T.val
    linewidth = s_line.val
    contrast = np.exp(- (np.pi * linewidth * T)**2)
    P = contrast * (np.cos(np.pi * delta_f * T)**2)
    ax2.clear()
    ax2.plot(delta_f, P, 'g-', linewidth=2)
    ax2.set_xlabel('Laser detuning Δf (Hz)')
    ax2.set_ylabel('Transition probability P')
    ax2.set_title(f'Ramsey Fringes (T = {T:.1f}s, linewidth = {linewidth:.1f} Hz)')
    ax2.grid(True)
    fig2.canvas.draw_idle()

s_T.on_changed(update)
s_line.on_changed(update)
```
